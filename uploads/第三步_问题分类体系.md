# 第三步：问题分类体系

**基于**：5月全量数据（60,802行）小样本分析（600条首轮问题）
**分析日期**：2026-06-02
**分类设计原则**：按资源/输出类型为第一维度组织大类，查找 vs 生成为二级维度拆子类；不强制合并不同意图，颗粒度优先。

---

## 一、分类体系总览

```
1. 教案 / 教学设计 / 学案
   ├─ 1a. 查找教案/教学设计/学案
   └─ 1b. 生成教案/教学设计/学案

2. 课件 / PPT（含交互式课件）
   ├─ 2a. 查找课件/PPT
   ├─ 2b. 生成课件/PPT（传统幻灯片）
   └─ 2c. 生成交互式课件（HTML/代码实现，教育向）

3. 试卷 / 试题 / 习题
   ├─ 3a. 查找试卷/真题/考卷
   ├─ 3b. 生成试卷/习题/变式题
   └─ 3c. 试卷/命题分析（分析命题规律、考点分布，非解题）

4. 知识点 / 讲义 / 复习资料
   ├─ 4a. 查找知识点/讲义/复习资料
   └─ 4b. 生成知识点/讲义/复习资料

5. 思维导图
   └─ 5. 思维导图生成

6. 学科知识问答
   └─ 6. 学科知识问答（概念/原理/考情解释）

7. 题目求解 / 解析
   └─ 7. 题目求解/解析（含完整题目材料）

8. 作文 / 写作文本
   ├─ 8a. 教学写作（作文/演讲稿/教学方案外文本）
   └─ 8b. 创意文学（小说/同人/言情）

9. 教学咨询
   └─ 9. 教学策略咨询/教学规划

11. 翻译
    └─ 11. 翻译（中英互译、语言转换）

12. 图片 / 视频 / 3D 生成
    └─ 12. 图片/视频/3D内容生成

13. 成绩 / 教学数据分析
    └─ 13. 学生成绩分析/教学质量报告

10. 非教学内容（噪声）
    ├─ 10a. 网址/入口导航
    ├─ 10b. 个人生活/娱乐/闲聊
    ├─ 10c. 一般知识查询（非学科）
    ├─ 10d. 产品功能询问
    ├─ 10e. 闲聊/乱输入/续问片段
    └─ 10f. 娱乐游戏生成（H5游戏，娱乐向）
```

**共 13 大类，23 个末级分类。**

> **结构说明**：
> - 交互式课件（HTML/代码）归入**课件大类（2c）**：代码是实现手段，课堂呈现才是本质，和传统PPT同属"可在课堂使用的视觉/交互载体"。
> - 知识问答（6）和题目求解（7）各自独立成大类：两者用户意图、输入形态、所需能力均不同，合并会掩盖各自的产品价值。
> - 娱乐游戏（10f）归入非教学内容：与产品定位不符，不建议主动承接。

## 二、各末级分类详细定义

### 1a. 查找教案/教学设计/学案

**定义**：用户要搜索或获取已存在的教案、教学设计、学案、说课稿、导学案等教学方案类文档，期望 AI 从资源库中检索。

**特征词**：教案、教学设计、学案、导学案、说课稿、学习任务单、帮我找/搜索/查找 + 教案|教学设计

**正例**：

1. "初中音乐人音版（简谱）☆ 乘着歌声的翅膀教案"
2. "高考 最后一课数学教案"
3. "初中化学沪教版整理与归纳教案"

**反例**：

1. "生成部编版历史新教材八年级下第18课...的教学设计" → **1b**（含生成动词）
2. "帮我检查这份公开课教学设计的逻辑是否清晰" → **9**（教学咨询，非查找）

**与现有场景关系**：属于「找资源」场景

---

### 1b. 生成教案/教学设计/学案

**定义**：用户要求 AI 创作新的教案、教学设计、学案、说课稿、导学案等教学方案文档。

**特征词**：生成/写/帮我写/设计 + 教案|教学设计|学案|导学案|说课稿；请根据...设计一份教学；教学目标/教学过程/板书设计（结构性词同时出现）

**正例**：

1. "生成部编版历史新教材八年级下第18课擘画中国梦宏伟蓝图的教学设计"
2. "写一篇高中数学北师大版对数的运算的教学设计"
3. "请根据课堂教学导学方案，生成一份导学案（物理·力·八年级）"

**反例**：

1. "初中化学沪教版整理与归纳教案" → **1a**（无生成动词，默认查找）
2. "帮我生成一个高三函数一轮复习公开课课件" → **2b**（课件不是教案）

**与现有场景关系**：属于「生成长文本」场景

---

### 2a. 查找课件/PPT

**定义**：用户要搜索或获取已存在的课件、PPT、幻灯片等可视化展示文件。

**特征词**：帮我找/搜索/查找 + 课件|PPT；有没有...课件；免费课件

**正例**：

1. "请帮我搜索沁园春雪的课件"
2. "帮我找一份鲁教版高中地理热力环流的复习课件"
3. "高考阅读理解A篇应用文课件"

**反例**：

1. "生成一个高三函数一轮复习公开课课件" → **2b**（明确是生成）
2. "课件" 如果有生成语境（上文有"帮我做"）→ **2b**

**与现有场景关系**：属于「找资源」场景

---

### 2b. 生成课件/PPT

**定义**：用户要求 AI 生成课件、PPT、幻灯片内容或大纲，通常指定学科、主题、页数。

**特征词**：生成/帮我做/制作 + 课件|PPT|幻灯片；课件.*生成；ppt.*生成；封面页/页面内容（逐页描述PPT结构）

**正例**：

1. "生成一个高三函数一轮复习公开课课件"
2. "帮我生成一个中考作文赏析并进行方法总结的课件PPT，背景用浅蓝色"
3. "PPT大纲细化版：《密度——从阿基米德的王冠到物质鉴别》" + 后附逐页内容描述

**反例**：

1. "请帮我搜索沁园春雪的课件" → **2a**（查找，非生成）
2. "请根据课堂教学导学方案生成教学设计" → **1b**（教案，非课件）

**与现有场景关系**：属于「生成长文本」场景

---

### 3a. 查找试卷/真题/考卷

**定义**：用户要搜索已有的试卷、历年真题、考卷、模拟卷等，通常指定地区、年份、科目。

**特征词**：查找/帮我找/搜索 + 试卷|真题|考卷|考题；近X年真题；XX省/市 + 试卷；历届真题；模考卷

**正例**：

1. "查找北京市2025年下学期高一数学期末考卷"
2. "找一下2024年全国甲卷"
3. "2025年济宁市高中招生物理试题"

**反例**：

1. "帮我出一套八年级历史...单元测试卷" → **3b**（生成，非查找）
2. "2026年高考地理考前20天冲刺讲义" → **4b**（讲义，非试卷）

**与现有场景关系**：属于「找资源」场景

---

### 3c. 试卷/命题分析

**定义**：用户要求分析已有试卷或一批题目的命题规律、考点分布、难度结构，目的是了解"题目怎么出的"而非解具体题目，也不是查找或生成试卷。

**特征词**：命题规律/趋势、考点分布、考情分析、考查方向、近X年命题趋势、分析这套试卷的考点、各省命题风格、XX题型在高考中如何考查

**正例**：

1. "分析近5年全国高考物理命题规律，主要考查哪些知识点，各知识点占比？"
2. "2025年广东中考数学和2024年有什么不同，新增了哪些考查点？"
3. "这份试卷的题型结构和难度分布合理吗？哪些知识点重复考查了？"

**反例**：

1. "如图所示，1/4圆弧轨道...求..." → **7**（解具体题目）
2. "查找2024年全国甲卷" → **3a**（查找试卷）
3. "帮我出一套命题规律和近年高考一致的试卷" → **3b**（生成试卷）

**与现有场景关系**：现有场景未覆盖，属于未命中场景（潜在新场景候选）

---

### 3b. 生成试卷/习题/变式题

**定义**：用户要求 AI 原创新题目、习题集、完整试卷或变式题，通常指定题型、数量、难度、知识点。

**特征词**：出一份/出一套/帮我出/生成/命制 + 试卷|题目|习题|练习题；变式题；组卷；指定数量+题型（如"30道选择题"）；难度系数

**正例**：

1. "帮我出一个高考化学的一轮复习资料，...15道近4年高考题（广东省优先）"
2. "命制一套2026年四川省高考物理高考前10天训练的高考适应性训练题（难度系数0.45）"
3. "帮我出八上生物这节课选择题30道，所有答案写在最下面"

**反例**：

1. "查找北京市2025年下学期高一数学期末考卷" → **3a**（查找，非生成）
2. "帮我搜索二次函数中用到斜率之积为-1的题" → **3a**（查找已有题目）

**与现有场景关系**：属于「生成长文本」场景（子场景）

---

### 4a. 查找知识点/讲义/复习资料

**定义**：用户要搜索已有的知识点总结、讲义、复习清单、知识梳理等学习资料，期望从资源库获取现成文档。

**特征词**：查找/找/搜索 + 知识点|讲义|清单|复习资料；有没有...知识清单；资源名+（无生成动词）

**正例**：

1. "帮我找一下圆锥曲线历届高考真题（含考情分析）"
2. "高中数学概率统计知识点"
3. "物理知识清单中考"

**反例**：

1. "需要你生成一份北师大版数学三年级下册知识点清单" → **4b**（有"生成"动词）
2. "整理出2026年高考英语听力词语" → **4b**（有"整理"动词，即生成）

**与现有场景关系**：属于「找资源」场景

---

### 4b. 生成知识点/讲义/复习资料

**定义**：用户要求 AI 生成知识点清单、专题讲义、复习资料、考前冲刺材料等，通常为原创内容。

**特征词**：必须含明确的**创作动词**（生成/写/帮我整理/帮我做/制作/请出）+ 知识点|讲义|复习资料|总结|清单|冲刺材料。**纯资源名称（无动词）默认归 4a**。

**正例**：

1. "把整个高中物理物理学史整理一篇word，哪个科学家发现了什么公式和规律"
2. "替我生成三个学生针对性全科七年级下册期末考前冲刺清单"
3. "帮我做一份高考化学考前回归教材的知识要点"

**反例**：

1. "2026年高考地理考前20天冲刺讲义" → **4a**（仅是资源名称，无创作动词，默认查找）
2. "物理知识清单中考" → **4a**（简短关键词，默认查找）
2. "生成部编版历史新教材八年级上第15课...的教学设计" → **1b**（教案，非知识点讲义）

**与现有场景关系**：属于「生成长文本」场景（子场景）

---

### 5. 思维导图生成

**定义**：用户要求生成知识结构图、思维导图、概念图等图形化知识组织工具。

**特征词**：思维导图、导图、知识结构图；生成/帮我画 + 导图；列提纲（有时也归此类）

**正例**：

1. "机械能及其守恒定律 思维导图"
2. "生成宇宙中的地球思维导图"
3. "按照沪科版初一下册数学目录生成思维导图，各章节重难点和解题技巧"

**反例**：

1. "帮我找一份初中化学思维导图" → **4a**（查找资源，不是生成）
2. "几何初步" → 无明确导图诉求，视上下文归入 **4a** 或 **7a**

**与现有场景关系**：属于「画导图」场景

---

### 2c. 生成交互式课件（HTML/代码实现，教育向）

**定义**：用户要求生成以 HTML/代码实现的交互式课堂内容，包括物理仿真、数学动画、知识点演示工具等，本质是课件的一种新型形态——代码是实现手段，核心是用于课堂展示或教学互动。

**特征词**：html/HTML + 教学|动画|仿真|互动|工具；以html形式输出 + 教学主题；制作.*动画.*教学；网页版教学工具；交互.*演示

**正例**：

1. "生成一个html，内容是初中数学勾股定理互动教学动画，引导学生主动探究"
2. "游乐场里的空中飞椅项目制作成动画以html形式输出，场景一相同稳定转速下不同绳长飞椅的运动"
3. "在空间里一个不带电的金属导体球A…绘制全空间的电场线分布过程，制作三维动画仿真，用HTML形式输出"

**反例**：

1. "帮我生成一个台球小游戏" → **10f**（娱乐游戏，非课堂内容）
2. "帮我做一个网页版计算器（html）" → 若无教学主题则视为通用工具，视内容判断

**与现有场景关系**：属于「画导图」或「写代码」场景，但更贴近课件场景；建议纳入课件大类统一入口

---

### 6. 学科知识问答

**定义**：用户直接提出学科知识问题，期待 AI 解释概念、分析原理、给出知识点说明，而非生成文档或查找资源。用户不提供具体题目，问的是知识本身。

**特征词**：是什么/为什么/怎么理解/有什么区别/简述 + 学科术语；学科术语 + 区别|原理|含义；直接给出物理/化学/语文等知识性问句；考情规律总结

**正例**：

1. "悬浊液乳浊液区别"
2. "直二面角生活实例"
3. "分泌蛋白和内在蛋白合成的起点都是游离的核糖体吗？"
4. "近三年有关化学试剂的保存方法的考查规律，试用三句话总结"

**反例**：

1. "总结人工智能的发展历程" → **10c**（非学科教育内容）
2. "帮我查找欧洲西部的高考真题" → **3a**（查找资源）
3. "如图所示，1/4圆弧轨道固定，R=0.2m...求..." → **7**（含完整题目参数的解题请求）

**与现有场景关系**：现有场景未覆盖，属于未命中场景（潜在新场景候选）

---

### 7. 题目求解/解析

**定义**：用户提供完整题目（含题干、选项或材料），要求 AI 作答、讲解解题思路、分析命题意图，或对阅读材料进行理解分析。区别于知识问答（6）的关键：**用户给出了具体题目**。

**特征词**：题目+选项（A/B/C/D）；下列分析正确的是；题干+求解；解析这道题；阅读理解文章+题目；完整题目材料（通常较长，含具体数值/人物/情节）

**正例**：

1. "七年级下册 招牌组同学要挑选一个最好的招牌，请你从以下三则招牌中选出一则并陈述理由。①曹操送货 ②故人庄饭店 ③三味书屋"
2. "2025年10月，受台风影响...由此可见 ①广大党员...②汛期青年党员...（政治选择题，含完整材料）"
3. "如图所示，1/4圆弧轨道固定，R=0.2m，传送带长L=5m...（物理计算题，含完整参数）"
4. "英语阅读理解原文（整段）配对应问题"

**反例**：

1. "悬浊液乳浊液区别" → **6**（知识问答，无题目结构）
2. "帮我出一道高中物理计算题" → **3b**（出题，非解题）
3. "帮我找关于牛顿第二定律的题目" → **3a**（查找，非解题）

**与现有场景关系**：现有场景未覆盖，属于未命中场景（潜在新场景候选）

---

### 8a. 教学写作（作文/演讲稿/非教学设计的文本）

**定义**：用户要求 AI 撰写作文、演讲稿、读后续写、说明文、寄语、总结报告等文本类内容。教师和学生均有此需求。

**特征词**：写一篇/帮我写 + 作文|演讲稿|寄语|报告|诗；续写；读后续写；作文题+写作要求；帮我为...写一篇文章

**正例**：

1. "帮我生成一份北京旅游攻略"
2. "写给闺蜜毕业信的800字真情流露"
3. "从去年入职成为班主任开始...我要写一篇教学案例（提供了完整草稿）"
4. "高三毕业班班主任寄语（简洁三维）"

**反例**：

1. "帮我生成一篇高中地理必修二人口分布的教学设计" → **1b**（教案，不是作文）
2. "帮我兄弟oc写一篇小说" → **8b**（创意文学）

**与现有场景关系**：属于「生成长文本」场景（子场景）

---

### 8b. 创意文学（小说/同人/言情）

**定义**：用户要求 AI 撰写同人小说、言情故事、创意剧情等非教育类创意文学内容。

**特征词**：oc/OC + 小说；言情；同人文；候府千金；帮我写一篇小说（含角色设定）；恋文；古代言情

**正例**：

1. "帮我兄弟oc写一篇小说 攻:南宫羽（弟弟）...受:南宫廖（哥哥）"
2. "生成一篇关于猫小九（男）...的一双人恋文，既要十分甜既要十分热烈"
3. "候府千金小姐下山，被团野宠翻了，古代言情小说，有结局，不要提纲，要内容，有马甲"

**反例**：

1. "写给闺蜜毕业信的800字" → **8a**（教学/生活写作，不是虚构文学）
2. "以《手工感》为题写一篇文章" → **8a**（作文写作）

**与现有场景关系**：现有场景均不适合处理，属于偏离产品定位的内容

---

### 11. 翻译

**定义**：用户要求进行中英文互译（或其他语言转换），包括单词、句子、段落、文章的翻译，以及语言润色、改写。

**特征词**：翻译、translate、用英文/中文说/写、怎么用英文表达、英文版、中文版、润色

**正例**：

1. "往而不可还者，亲也；至而不可加者，年也。翻译"
2. "用英语翻译一下爱与回忆永恒"
3. "find oneself doing sth（释义请求，实质是词汇翻译）"

**反例**：

1. "帮我写一篇英语演讲稿" → **8a**（英语写作，不是翻译）
2. "初中英语译林版教学设计" → **1a**（查找教案）

**与现有场景关系**：属于「生成长文本」场景子场景，建议独立入口

---

### 12. 图片/视频/3D生成

**定义**：用户要求生成图片、配图、视频内容或3D模型。与2c（交互式课件）的区别：这里的核心诉求是**视觉素材本身**，而非用于课堂交互的功能性工具。

**特征词**：生成一张图片、配图、帮我做一张、生成视频、生成3D、图片素材、画一张

**正例**：

1. "帮我生成一张20世纪40-50年代苏联工业面貌的图片"
2. "请根据文档完成配图"
3. "生成一个外接球的3D模型，可以旋转查看"

**反例**：

1. "生成一个html，内容是初中数学勾股定理互动教学动画" → **2c**（交互式课件）
2. "生成宇宙中的地球思维导图" → **5**（思维导图）

**与现有场景关系**：现有场景未覆盖，属于未命中场景（潜在新场景候选）

---

### 13. 成绩/教学数据分析

**定义**：用户提供或描述学生成绩数据、考试结果，要求生成成绩分析报告、班级质量分析、学生薄弱点诊断等数据驱动的教学分析内容。

**特征词**：成绩分析、质量分析、班级分析报告、平均分/及格率/优秀率、成绩数据、学情分析（含数据）、各科成绩

**正例**：

1. "根据这次期末考试的成绩数据，帮我生成一份班级质量分析报告"
2. "我们班数学平均分72分，及格率85%，优秀率23%，帮我分析主要问题"
3. "上传了学生期中考试成绩表，帮我找出各科薄弱的学生"

**反例**：

1. "帮我生成高三班主任质量分析" → **8a**（写作，没有具体数据驱动）
2. "分析近5年高考命题趋势" → **3c**（命题分析，不是学生成绩）

**与现有场景关系**：现有场景未覆盖，属于未命中场景（潜在新场景候选）

---

### 9. 教学策略咨询/教学规划

**定义**：教师就教学方法、公开课设计思路、备课建议、教研规划等提出咨询性问题，期待 AI 给出专业建议而非生成文档；或请 AI 制定教学计划/复习计划。

**特征词**：你有没有什么建议；如何让...更好；我是一名教师...怎么；教学计划；复习计划；有何建议；教学规划；备考方案；如何提升

**正例**：

1. "我是一名高中地理教师，我需要上一节公开课，你有没有一些建议，比如学生活动或者AI的应用"
2. "以下是一篇新闻。你觉得更适合改编成一篇阅读理解还是语法填空？"（教师寻求命题建议）
3. "我是周课老师，帮我制定一份数学北师大八年级下册从第四单元开始到期末的十周教学计划"

**反例**：

1. "生成一个高三函数一轮复习公开课课件" → **2b**（生成课件，不是咨询）
2. "近三年化学高考题有关实验安全的考情，用三句话总结" → **7a**（知识问答，不是策略咨询）

**与现有场景关系**：现有场景未覆盖，属于未命中场景（潜在新场景候选）

---

### 10a. 网址/入口导航

**定义**：用户直接输入网站名称或平台名，期望获取链接或跳转到对应网站。这是最典型的"把 AI 当搜索引擎"行为。

**特征词**：网址、入口、官网、链接、打开 + 平台名；抖音/百度/哔哩哔哩 + 网址；www.开头

**正例**：

1. "百度网址"（高频重复）
2. "抖音网址"（高频重复）
3. "给我Bing的网址的链接"

**反例**：

1. "帮我打开好一三的网址链接" → 仍归 **10a**（含"帮我"也是导航诉求）
2. "学科网里面的这个组卷网在哪里？" → **10d**（产品功能询问）

**与现有场景关系**：不适合设场景，属于产品入口误用，建议引导拦截

---

### 10b. 个人生活/娱乐/闲聊

**定义**：用户提出与教育无关的个人生活问题，包括健康咨询、电子产品问题、娱乐推荐、情感倾诉等。

**特征词**：膝关节/相机/手机（非教学设备）咨询；推荐歌曲；几点了；塔罗牌；朋友决裂；王者荣耀；特步跑鞋

**正例**：

1. "最近几个月，在起身时，左膝关节常发出卡卡的声音"（健康咨询）
2. "刚买了一个小相机，用充电器充电，可是刚开始充电，它就开始发热发烫了"
3. "华为watchPro和标准版的区别"

**反例**：

1. "你好" → **10e**（闲聊/乱输入）
2. "好气啊，觉得自己不够聪明，没能获取老师的关注" → **10b**（个人情感）

**与现有场景关系**：不适合设场景，属于偏离产品定位的用户群

---

### 10c. 一般知识查询（非学科）

**定义**：用户提出非教学学科的知识性问题，如历史事件、AI技术发展、GDP数据等，期待直接回答，但不属于中小学教育场景。

**特征词**：AI发展历程、GDP排名、法律问题、非教学领域的概念解释；总结人工智能发展

**正例**：

1. "2025年全年GDP排名"
2. "总结人工智能的发展历程"（作为一般知识，非教学目的）
3. "请问可以说，A对B做出一个行为，B假如没有第一时间检举，就默许了A的行为..."（法律逻辑）

**反例**：

1. "人工智能的发展历程（作为教学课件/教案素材）" → 需结合上下文，若是备课需求则归 **1a/4b**
2. "悬浊液乳浊液区别" → **7a**（初中化学学科知识）

**与现有场景关系**：不适合设场景，属于通用 AI 行为

---

### 10d. 产品功能询问

**定义**：用户询问小博士产品能做什么、如何使用某功能、是否支持某操作。

**特征词**：你能...吗、可以判卷吗、组卷网在哪里、如何搜索id、怎么下载

**正例**：

1. "你能判卷吗"
2. "学科网里面的这个组卷网在哪里？"
3. "如何搜素id"

**反例**：

1. "帮我找一份课件" → **2a**（具体的使用请求，不是功能询问）

**与现有场景关系**：不适合设内容场景，适合做功能引导

---

### 10e. 闲聊/乱输入/续问片段

**定义**：用户输入问候语、单字母/无意义字符、依赖上下文才能理解的续问片段，脱离上下文无法独立判断意图。

**特征词**：你好；B/A/C（单字母）；怎么写？（无上文）；在吗

**正例**：

1. "B"（单字母，疑为对上一题的答案确认或续问）
2. "怎么写？是否有原题"（无法确定所指题目）
3. "你好"

**与现有场景关系**：不适合设场景

## 四、分类规则（判断树）

按优先级从高到低依次判断，命中即停止：

```
Step 1 — 快速排除噪声
  ├─ 是否为纯网址请求？（含"网址""官网""打开www."等，无教学词） → 10a
  ├─ 是否为单字符/极短无意义输入 或 脱离上下文的续问？ → 10e
  └─ 否则 → Step 2

Step 2 — 判断资源类型（是否有明确的教学内容类型词）
  ├─ 含「教案/教学设计/学案/导学案/说课稿」
  │   ├─ 含生成动词（生成/写/帮我写/设计/请根据...生成） → 1b
  │   └─ 无生成动词 → 1a
  ├─ 含「课件/PPT/幻灯片」
  │   ├─ 含生成动词（生成/做/帮我做） → 2b
  │   └─ 无生成动词 → 2a
  ├─ 含「试卷/真题/考卷/模拟卷」且含出题/组卷动词 → 3b
  ├─ 含「试卷/真题/考卷」无出题动词 → 3a
  ├─ 含「出题/出一道/出一份/出一套/命题/命制/变式题」+ 题量/难度词 → 3b
  ├─ 含「知识点/讲义/复习资料/清单」
  │   ├─ 含生成动词 → 4b
  │   └─ 无生成动词 → 4a
  ├─ 含「思维导图/导图」 → 5
  └─ 否则 → Step 3

Step 3 — 判断输出类型（交互课件/问答/写作/游戏）
  ├─ 含「html/HTML/动画/仿真」且主题为教学知识 → 2c（交互式课件）
  ├─ 含「游戏/跑酷/台球/塔防/射击/吞噬/GTA」等纯娱乐词 → 10f
  ├─ 含「翻译/translate/用英文/用中文/怎么说」 → 11
  ├─ 含「生成图片/生成视频/配图/3D模型/画一张」 → 12
  ├─ 含「命题规律/命题趋势/考点分布/考情分析/命题风格」 → 3c
  ├─ 含「成绩分析/质量分析/平均分/及格率/班级成绩/学情分析」且含数据 → 13
  ├─ 含具体题目结构（选项A/B/C/D + 题干材料，或完整物理/数学/语文题目参数） → 7
  ├─ 含学科术语 + 知识性问句（是什么/为什么/区别/原理/考情） → 6
  ├─ 含「作文/演讲稿/续写/寄语/毕业信/攻略/介绍词」等写作词 → 8a
  ├─ 含「小说/oc/言情/同人」等创意词 → 8b
  ├─ 含「有何建议/教学计划/备课建议/你有没有一些建议」 → 9
  └─ 否则 → Step 4

Step 4 — 非教学分类
  ├─ 个人生活、健康、电子产品、娱乐、情感相关 → 10b
  ├─ 非学科知识问答（AI、法律、财经等） → 10c
  ├─ 产品能力询问（你能...吗/在哪里/怎么用） → 10d
  └─ 其他无法归类 → 10e
```

## 五、易混淆分类对与区分要点

| 混淆对 | 区分关键 |
|--------|---------|
| **1a 查找教案 vs 1b 生成教案** | 有"生成/写/帮我写/设计"等创作动词 → 1b；无动词或仅有"找/搜索" → 1a |
| **2a 查找课件 vs 2b 生成课件** | 同上，看动词；逐页描述PPT结构（如"封面页/P1/P2"）= 2b |
| **3a 查找试卷 vs 3b 生成试题** | "查找/有没有/找一下" → 3a；"出一份/命制/生成/组卷" → 3b |
| **4a 查找讲义 vs 4b 生成讲义** | 同上，短关键词（如"物理知识清单中考"无动词）默认 4a |
| **1b 教案 vs 2b 课件** | 明确提"PPT/课件/幻灯片" → 2b；提"教案/教学设计" → 1b；两者都提 → 以主要诉求为准 |
| **2b 传统课件 vs 2c 交互式课件** | 要求输出 HTML/代码/动画 且主题是教学知识 → 2c；要求输出幻灯片内容/逐页文字 → 2b |
| **2c 交互式课件 vs 10f 娱乐游戏** | HTML主题是教学知识（勾股定理/物理仿真）→ 2c；主题是纯游戏娱乐（跑酷/台球）→ 10f |
| **3b 生成试题 vs 6 知识问答** | "出一道题" + 指定题型/难度 → 3b；"某知识点是什么/如何解释" → 6 |
| **6 学科知识问答 vs 7 题目求解** | 用户自己提供完整题目（含选项/材料/参数） → 7；开放性知识问题 → 6 |
| **7 题目求解 vs 3a 查找试卷** | 用户粘贴了完整题目内容期待解析 → 7；只说"找某某试卷/真题" → 3a |
| **3c 命题分析 vs 7 题目求解** | 分析"这类题在高考怎么考/命题规律是什么" → 3c；解具体一道题 → 7 |
| **3c 命题分析 vs 3a 查找试卷** | 要分析/解读命题模式 → 3c；要拿到试卷文件本身 → 3a |
| **13 成绩分析 vs 9 教学咨询** | 有具体成绩数据驱动（平均分/及格率/班级数据） → 13；泛化的教学策略咨询 → 9 |
| **11 翻译 vs 8a 教学写作** | 核心诉求是语言转换（给定原文求译文）→ 11；核心诉求是生成全新文本内容 → 8a |
| **12 图片生成 vs 2c 交互课件** | 要一张静态图片/视频素材 → 12；要可在课堂展示的HTML交互动画 → 2c |
| **8a 教学写作 vs 1b 生成教案** | 产出是教师用的教学方案 → 1b；产出是学生/通用文本（作文/信件/寄语） → 8a |
| **10a 网址导航 vs 10d 产品功能询问** | 要第三方网站链接 → 10a；询问平台自身功能/位置 → 10d |
